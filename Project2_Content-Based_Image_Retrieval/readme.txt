Instructions:
The project contains two parts: CLI (command line interface) and GUI (Graphical User Interface):
CLI self print tasks from 1-5 and two extensions in the terminal (hard coded target images).
	*For example, Task 1 would print:
		The top 3 results for 0164 matching are:
		pic.0080.jpg
		pic.1032.jpg
		pic.0461.jpg

GUI gives user options to choose what task or extension to run and customized matching size, and returns the best matche results.
	The tasks and extensions including:
	Task1: Baseline Matching;
	Task2: Histogram Matching;
	Task3: Multi-histogram Matching;
	Task4: Texture and Color;
	Task5: Custom Design;
	Extension1: Garbor and RGB Matching:
	Extension2: HSV and RGB Matching.

	*Please see Wiki for more content.




This project contains three extensions including:
	Extension0: GUI;
	Extension1: Gabor Texture and RGB and Color Histogram;
	Extension3: HSV and RGB Multi-histogram.


To test executable for GLI, please run readfiles.exe
To test executable for GUI, please run GUI.exe
